Jquery-hy_slide多功能滚动插件 v1.0 参数说明
参数名 参数值(默认值可以省略) 特别说明
effect scrollY (默认)：垂直回转式滚动 回转式滚动是指移动到末尾处后反方向滚动回到开始处，循环式滚动则是指移动到末尾处后保持方面不变回到开始处
scrollLoopY：垂直循环式滚动
scrollX：水平回转式滚动
scrollLoopX：水平循环式滚动
fade：渐变式切换
autoPlay true (默认) , false：是否自动执行  
corotation true (默认) , false：是否正转 由下到上、由右到到左为正转
speed fast, normal (默认), slow, 正整数：动画执行速度  
timer fast, normal (默认), slow, 正整数：动画执行间隔  
show 正整数：一屏里展示的信息数量，默认展示1个，该参数对fade无效  
step 正整数：每次动画移动的信息数量，默认移动1个，该参数对fade无效  
classTr jQuery类选择器，默认为"hySlideTrigger"：触发器父级class  
classCon jQuery类选择器，默认为"hySlideContent"：滚动器父级class  
btnPrev jQuery类选择器，默认为"prev"：上一个按钮class  
btnNext jQuery类选择器，默认为"next"：下一个按钮class  
loop true (默认) , false：是否循环执行动画  
slight 正整数，默认为0：项目微调，单位px 对滚动器宽高进行细微调整

动画代码：

$('#demo1').hy_slide({
    effect : 'scrollY',
    autoPlay : true,
    corotation : true,
    speed : 1000,
    timer : 3000,
    show : 3,
    step : 2,
    classTr : 'hySlideTrigger',
    classCon : 'hySlideContent',
    btnPrev : 'prev',
    btnNext : 'next',
    loop : true,
    slight : 6
});
